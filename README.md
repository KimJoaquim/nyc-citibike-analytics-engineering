# Nome do Projeto (ex: NYC Citi Bike Analytics Pipeline)

## 📋 Sobre o Projeto
Descrição curta e forte: "Pipeline de dados end-to-end para análise de mobilidade urbana utilizando dados públicos do Citi Bike (NYC). Transformação de milhões de registros em insights acionáveis para otimização logística."

## 🎯 Objetivo de Negócio
- Identificar padrões de uso (picos de demanda por hora e estação)
- Auxiliar no planejamento de frota e logística urbana

## 🛠️ Tecnologias Utilizadas
- BigQuery (Data Warehouse)
- SQL (Standard SQL)
- Looker Studio (Visualização)
- Google Cloud

## 🏗️ Arquitetura
- Camada Raw → Staging → Marts (modelagem dimensional)
- ...

## 📊 Principais Insights
- Pico de uso entre 17h-19h (comportamento corporativo)
- ...
